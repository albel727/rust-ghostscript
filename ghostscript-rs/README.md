ghostscript-rs
==============

High-level Rust API bindings for the Ghostscript PS/PDF interpreter library.

Usage
=====

Put this in your `Cargo.toml`.
```
[dependencies]
ghostscript = "0.1.0"
```

See module-level documentation for the library overview.

Examples
========
See [examples](examples) subdirectory.
